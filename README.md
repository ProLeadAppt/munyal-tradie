# Munyal — AI receptionist for tradies

Landing page for Munyal Pty Ltd. Single static HTML file. No build step.

## Deploy

This site is deployed on Vercel. Push to `main` and Vercel auto-deploys.

## Local preview

```bash
python3 -m http.server 8000
# open http://localhost:8000
```

## File structure

- `index.html` — the entire site. All copy comes from `/root/munyal-content/landing-page-copy-v1.md` v1.2.
